# Mambaによる動き予測を用いたSAM2ベースの物体追跡

## 概要

SAM2はMeta社が開発したセグメンテーションモデルであり、ビデオに適応可能なことから人物追跡も実現している。一方で、このモデルはセグメントに重きを置いているため、人物追跡の手法としてみると改善の余地が存在している。そこで本研究では、SAM2に学習ベースの追跡手法であるMambaを導入する手法を提案・考案する。
